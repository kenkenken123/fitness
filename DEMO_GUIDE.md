# AI器材识别功能演示指南

## 演示步骤

### 1. 启动应用
```bash
# 启动后端服务
cd backend
dotnet run

# 启动前端服务
cd frontend
npm run dev
```

### 2. 登录应用
1. 打开浏览器访问 `http://localhost:3000`
2. 选择"注册"标签页，输入用户名、邮箱和密码进行注册
3. 或者选择"登录"标签页，输入用户名和密码进行登录
4. 登录成功后会自动跳转到首页

### 3. 创建训练环境
1. 点击底部导航栏的"训练环境"
2. 点击右下角的"+"按钮
3. 输入环境名称（如"家庭健身房"）
4. 点击"确认添加"

### 4. 使用AI识别功能
1. 点击刚创建的训练环境进入详情页
2. 在器材列表页面，点击"AI识别"按钮
3. 在弹出的对话框中：
   - 点击"拍照"按钮使用摄像头拍摄器材
   - 或点击"上传"按钮选择已有图片
4. 拍摄/选择图片后，点击"开始识别"
5. 等待2-3秒，查看AI识别结果
6. 勾选要添加的器材
7. 点击"添加选中器材"完成添加

### 5. 验证结果
1. 返回器材列表页面
2. 确认新识别的器材已添加到列表中
3. 可以继续使用AI识别添加更多器材

## 演示要点

### 功能亮点
- **拍照识别**: 实时摄像头拍摄，支持多种器材识别
- **批量处理**: 一次识别多个器材，提高效率
- **智能推荐**: 自动识别器材类型和重量
- **用户友好**: 简洁的界面和流畅的操作体验

### 技术特色
- **响应式设计**: 适配移动端和桌面端
- **实时反馈**: 识别过程中显示加载状态
- **错误处理**: 完善的异常处理机制
- **可扩展性**: 易于集成真实AI服务

## 测试场景

### 场景1: 拍照识别
1. 准备一些健身器材（哑铃、杠铃等）
2. 使用AI识别功能拍照
3. 验证识别准确性和置信度

### 场景2: 上传图片
1. 准备器材图片文件
2. 使用上传功能识别
3. 对比不同图片的识别效果

### 场景3: 批量添加
1. 拍摄包含多个器材的图片
2. 选择部分器材进行添加
3. 验证批量添加功能

## 注意事项

1. **浏览器兼容性**: 建议使用Chrome或Safari浏览器
2. **摄像头权限**: 首次使用需要授权摄像头访问权限
3. **网络连接**: 确保网络连接稳定
4. **图片质量**: 清晰的图片能提高识别准确率

## 故障排除

### 常见问题
1. **摄像头无法启动**: 检查浏览器权限设置
2. **识别失败**: 检查网络连接和后端服务状态
3. **图片上传失败**: 检查图片格式和大小

### 调试方法
1. 打开浏览器开发者工具查看控制台错误
2. 检查后端服务日志
3. 验证API接口响应
