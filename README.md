# Fitness App

一个用于辅助健身、记录训练的 Web 应用，采用前后端分离架构。

## 功能特性

- **用户系统:** 支持游客模式自动注册和登录。
- **训练记录:** 创建和查看包含多个训练项目的详细训练日志。
- **地址管理:** 对训练地点进行增、删、改、查管理。
- **现代化UI:** 采用暗色主题和移动优先的 App 设计风格。

## 技术栈

- **后端:** .NET Core, Furion, MySQL
- **前端:** Next.js (React), Material-UI (MUI)

## 如何运行

### 运行后端

1. 进入 `backend` 目录。
2. 运行 `dotnet run`。

### 运行前端

1. 进入 `frontend` 目录。
2. 运行 `npm install` 安装依赖。
3. 运行 `npm run dev`。
